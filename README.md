# ros2-control-vcan-motor-demo

A ROS 2 Humble demo that connects `diff_drive_controller` to two virtual motors over a WSL2
SocketCAN `vcan0` interface.

## Environment

- WSL2 Ubuntu 22.04
- ROS 2 Humble
- `ros2_control`, `ros2_controllers`, `ros2_socketcan`

Implementation and run instructions are added as the demo is built.
